# Prompt UX

A single-page app where you browse 10 startup design themes and copy a ready-to-use design prompt for each one.

The idea: stop guessing what your landing page should look like. Pick a theme that fits your product, copy the prompt, and hand it to an AI or designer.

## Themes

SaaS Minimal · AI & Deep Tech · FinTech & Trust · E-Commerce · EdTech · Health & Wellness · Developer Tools · Sustainability · Social & Community · Gaming

Each theme includes its design origins, where it's typically used, font pairing rationale, and a full copyable prompt.

## Running locally

```bash
npm install
npm run dev
```

## Stack

Simply Vite + React, no backend, plain CSS.
